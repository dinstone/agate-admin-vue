## 介绍

`agate-admin-vue`是Agate网关管理前端项目，其后端项目为[Agate](https://github.com/dinstone/agate)。

`agate-admin-vue`基于[vue-element-plus-admin](https://kailong110120130.gitee.io/vue-element-plus-admin)模板开发。

`vue-element-plus-admin` 是一个基于 `element-plus` 免费开源的中后台模版。使用了最新的`vue3`，`vite`，`TypeScript`等主流技术开发，开箱即用的中后台前端解决方案，可以用来作为项目的启动模版，也可用于学习参考。


`vue-element-plus-admin` 在线预览 :

- [vue-element-plus-admin](https://element-plus-admin.cn/) - 完整版 github 站点
- [vue-element-plus-admin](https://kailong110120130.gitee.io/vue-element-plus-admin) - 完整版 gitee 站点

## 安装和使用

- 获取代码

```bash
git clone https://github.com/dinstone/agate-admin-vue.git
```

- 安装依赖

```bash
cd agate-admin-vue

pnpm install
```

- 运行

```bash
pnpm run dev
```

- 打包

```bash
pnpm run build:pro
```
